# 🤖 Rule-Based AI Chatbot

## 📌 Project Description

A simple Rule-Based AI Chatbot built using Python. The chatbot responds to predefined user inputs using conditional statements (`if-else`) and runs continuously until the user exits the conversation.

## 🎯 Objective

To understand the basics of Artificial Intelligence by creating a chatbot that can interact with users using predefined rules and logical decision-making.

## ✨ Features

* Responds to greetings
* Answers basic questions
* Handles exit commands
* Uses `if-else` statements
* Runs in a continuous loop

## 🛠 Technologies Used
